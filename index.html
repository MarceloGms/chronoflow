What is Chronoflow ?

Chronoflow is a software that suggests personalized schedules based on student preferences. It is meant for all students who need help creating schedules and who are dissatisfied with the manual process of selecting the combination of class periods. Without having to plan schedules manually, this software serves as a tool to help you plan the ideal time!
